Parts of a Computer

PC Components Guide

CPU (Central Processing Unit)

The CPU is the brain of the computer, handling all calculations and instructions.

RAM (Random Access Memory)

RAM temporarily stores data for quick access, speeding up the computerâ€™s performance.

GPU (Graphics Processing Unit)

The GPU renders images, videos, and graphics, essential for gaming and 3D work.

HDD (Hard Disk Drive)

HDDs provide large storage capacity for OS, files, and applications.

SSD (Solid State Drive)

SSDs are faster than HDDs, improving boot time and application speed.

Power Supply Unit (PSU)

The PSU converts electricity into usable power for all computer components.

Motherboard

The motherboard connects all components and allows communication between them.

Cooling System

The cooling system prevents overheating using fans, heatsinks, and liquid cooling solutions.

PC Assembly & Disassembly Guide

Step 1: Prepare the Workspace

Ensure a clean, static-free area and gather all necessary tools.

Step 2: Install the CPU

Open the CPU socket, align the processor, and secure it in place.

Step 3: Insert RAM

Align RAM sticks with motherboard slots and press them down firmly.

Step 4: Attach Storage Drives

Mount HDD/SSD into drive bays and connect SATA cables.

Step 5: Install Power Supply

Secure the PSU in the case and connect power cables to components.

Step 6: Add GPU

Insert the graphics card into the PCIe slot and secure it.

Step 7: Connect Cables

Attach motherboard, power, and data cables as per the manual.

Step 8: Power On & Test

Turn on the system, check for display output, and enter BIOS.
